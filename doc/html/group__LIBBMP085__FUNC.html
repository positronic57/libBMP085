int BMP085_initSensor ( BMP085 *  sensor,
int *  I2Cbus,
unsigned char  I2CAddress,
overSampling  oss 
)

Sensor initialization.

The function assigns the file descriptor for accessing I2C bus to the sensor. Defines the I2C address and the mode of operation. After that it reads the sensor calibration table used later on for calculating the temperature and pressure values. Call this function first, before any other function related to the sensor.

Parameters
[in,out]sensorpointer to the structure that represents the sensor.
[in]I2Cbuspointer to the file descriptor of the I2C bus present on the system.
[in]I2CAddressaddress of the sensor on the I2C bus.
[in]ossdefining the over sampling mode of the sensor for the pressure measurement.
Returns
int returns 0 for successful operation, 1 for failure.
void BMP085_printCalibrationTable ( BMP085 *  sensor)

The function prints the calibration table of the BMP085 sensor on the standard output.

The function should be called after BMP085_initSensor() function. Otherwise it will print random values for the coefficients.

Parameters
[in]sensorpointer to the structure that represents the sensor.
int BMP085_takeMeasurement ( BMP085 *  sensor)

Starts the process of temperature and pressure measurement.

After the raw temperature and pressure data are retrieved from the sensor, the function calculates the temperature values in degrees Celsius and barometric pressure in [Pa].
