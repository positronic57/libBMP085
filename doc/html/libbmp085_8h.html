int BMP085_Altitude ( double  lastPressureReading,
double  baseLinePressure 
)

Calculates the relative altitude changes in [m] based on pressure difference between two measurement points.

If the average sea level pressure is used as a reference point, the function will return the altitude above sea level in [m].

Parameters
[in]lastPressureReadingLast pressure reading.
[in]baseLinePressureReference pressure for calculating the relative altitude changes.
Returns
int The relative altitude change due to difference in pressure values.
